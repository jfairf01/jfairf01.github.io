Personal Projects:

Facial Recognizer; https://facial-recognizer.herokuapp.com

Role: Full Stack (December 2017)

      This project was in association with my Object Oriented Programming for Graphical User Interface class. This project was inspired as a proof of concept from an article I read that contemplated the idea that one day our facial recognition would be used instead of a passport for travel. If this were to happen (and currently most international travel includes a portrait of the traveller), then if these airports were able to cross-reference a database of missing persons reports using a facial recognition API I have used here, it may help to reduce the incidents of human trafficking.

Technologies Used: Heroku, PostgreSQL, Flask, Jinja2, Semantic UI, Kairos (Facial Recognition API), Cloudinary (cloud-based photo storage API), Python (Web Scraper)

University Reservation System; http://urs-tufts.herokuapp.com/

Role: Full Stack (January 2017-May 2017)

      This project was in association with my Web Engineering class. I worked with a group of three to create a new reservation system for Tufts University that allows users to easily reserve spaces and equipment on campus by routing requests and keeping track of security clearance to quickly, safely, and effectively allow groups on campus to reserve materials. Only Spaces works as it was a proof of concept project. (For a reservation, select the Tufts Medford/Somerville Campus and a Classroom as the Space Type)

Technologies Used: Heroku, PostgreSQL, Flask, Jinja2, Semantic UI


SquirHunt; https://squirrelhunt.herokuapp.com/

Role: Backend Development(November 2016)

      This project was in association with my Web Programming class. I worked with a group of four to create a website application that allows users to sign up or log in to post photos of squirrels including the location of their sighting.
